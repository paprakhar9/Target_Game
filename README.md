# Target Game

An interactive web game where you click on a moving target to score points. The game is responsive and works on all screen sizes.

## How to Play

1. Open `index.html` in your web browser.
2. Click the red target circle as it appears in random locations within the game area.
3. Each successful click increases your score by 1.
4. The target will move to a new random position after each click.
5. Try to score as many points as possible!

## Features
- Responsive design for desktop and mobile
- Animated target with hover and click effects
- Real-time score tracking

## Files
- `index.html` — Main HTML file
- `style.css` — Game styling and responsive layout
- `game.js` — Game logic and interactivity

## Running Locally
Just open `index.html` in any modern web browser. No installation or server required.
